install cub3D game
```
git clone https://github.com/Bouhrir/cub3d_game.git lo3ba
```

macOS Users:
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
and:
```
brew install glfw
```
run:
```
cd lo3ba/
```
```
make bonus && ./cub3d_bonus map/map1.cub
```
